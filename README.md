First step:
Create a folder where your project will be located
Install "npm" then run in your terminal `npm init` >> creates a file package.json of our new project
Then install the discord.js library using `npm install discord.js` that should install a folder of the most recent version of discord.js

Second step:
Create your index.js file, organize your functions in different folders ex. `Functions/help.js` this will help in the long run, keep your index.js organized

Third step:
You can follow the way I have organized mine and how I'm exporting the functions back to `index.js`

Final step:
Run your bot with `node index.js`
